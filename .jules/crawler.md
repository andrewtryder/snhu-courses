## 2024-07-03 - [SEO: Sitelinks Search and Breadcrumb Schemas]
**Learning:** When H1s and other visible markup are tightly integrated into the UI (e.g., specific layouts/fonts on the home page), we can achieve high-value indexing and rich result capabilities strictly through invisible JSON-LD structured data. We added `SearchAction` to the homepage and `BreadcrumbList` to dynamic course pages.
**Action:** Use schema markup as the default tool for SEO enhancements when visual layout changes carry a risk of UI disruption.
